 Basic Calculator 🧮  

## Description  
This is a simple Basic Calculator built using Python. It allows users to perform basic arithmetic operations, including:  
- Addition (+)  
- Subtraction (-)  
- Multiplication (*)  
- Division (/)  

## Features  
✅ User-friendly input system  
✅ Handles division by zero with an error message  
✅ Runs continuously until the user chooses to exit  

## Usage  
1. Run the Python script.  
2. Enter two numbers.  
3. Choose an arithmetic operation.  
4. View the result.  
5. Repeat until you decide to exit.  

## Example Output  
Enter first number: 10
Choose operation (+, -, *, /): /
Enter second number: 0
Error: Division by zero is not allowed.



🎯 Number Guessing Game

📌 Project Overview
The Number Guessing Game is a simple Python-based game where the player tries to guess a randomly chosen number between 1 and 100. The program provides hints after each incorrect guess, helping the player reach the correct answer.

🔧 How It Works:
The program generates a random number between 1 and 100.
The player enters a guess.
The program responds with hints:
Too high? Try a lower number.
Too low? Try a higher number.
The game continues until the correct number is guessed. 🎉

🚀 Technologies Used:
Python
Random module for number generation
Loops & Conditional Statements for game logic
User Input Handling

🖥️ How to Run the Game: 
Clone the repository or download the Python script.

Run the script in any Python environment:
python number_guessing_game.py

Start guessing the number!
